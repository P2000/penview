__init__(self, p=':memory:', nv=1)
(Constructor)

source code 

initiate a new experiment

(p defaults to ":memory:" if not specified:
the database is lost when python quits - good for testing)

Parameters: p is the filesystem path where the experiment will be stored nv is the number of parameters (y-values) in the data-set
